#### My code

```python
def solution(participant, completion):
    participant.sort()
    completion.sort()
    for p, c in zip(participant, completion):
        if p != c:
            return p
    return participant[-1]
```

#### Using `Counter` method
 
 ```python
import collections

def solution(participant, completion):
    return list(collections.Counter(participant) - collections.Counter(completion))[0]
 ```
